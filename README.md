# AI Hospital System

## Description

AI Hospital System is a web-based application that performs basic health analysis using predefined logic. It allows users to input data and receive predictive results through an integrated system.

## Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB

## Setup

1. Clone the repository
   git clone https://github.com/xjazzbladezz/AI-Hospital2.git

2. Install dependencies
   cd backend
   npm install

cd ../frontend
npm install

3. Run the application
   Start backend: npm start
   Start frontend: npm start

## Configuration

Create a `.env` file in the backend folder:

PORT=5000
MONGO_URI=your_connection_string
